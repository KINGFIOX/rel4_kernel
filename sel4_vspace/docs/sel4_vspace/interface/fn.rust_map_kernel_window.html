构建reL4的内核页表,主要完成了PSpace和KERNEL ELF两段虚拟地址空间的映射

其中PSpace是对整个物理地址空间的线性映射，KERNEL ELF是对内核代码的再一次映射

reL4的地址空间如下图所示（来源：seL4/include/arch/riscv/arch/64/mode/hardware.h），左侧为虚拟地址空间，右侧为物理地址空间：
